About Us......


Hello, world! Our names are Baji Tumendemberel and Will Zhu and flatNotes is our final project for CS50. To use this site, either write freely in the textbox or copy and paste text. The text will be submitted either at the end of our two minute timer or you can press submit if you don't need that much time. You can use the journals page to keep track of your past journal entries in chronological order.

Once you have submitted the journal entry, the text will be read by a sentiment analysis function found in this Python library. This will return emotional values corresponding to happy, sad, fear, surprise, and angry. Using these values, we are able to display your emotional composition. We are also working on adding the functionality to create a Spotify playlist using the emotional values as metrics to select for songs within your top 100 most recent liked songs, so be on the lookout!
